# Burnout Garden

Burnout Garden is a React and Tailwind CSS web app that helps users track daily habits and burnout risk through an interactive garden visualization.

The app asks users simple daily questions about sleep, work, exercise, social life, screen time, stress, and other lifestyle habits. Based on the answers, a tree grows or weakens to visually represent the user's wellbeing.

## Features

- Daily check-in form
- Interactive tree visualization
- Animated branch growth and decline
- Watering can animation
- Weekly summary and insights
- Local storage support
- Responsive design
- Built with React and Tailwind CSS

## Tech Stack

- React
- Tailwind CSS
- Framer Motion
- Local Storage
